## 📐 Общие параметры

### Цветовая палитра

```css
/* Основные цвета */
--primary-purple: #8B5CF6     /* Фиолетовый акцент */
--primary-blue: #3B82F6       /* Синий акцент */

--text-primary: #F8FAFC       /* Основной текст */
--text-secondary: #94A3B8     /* Вторичный текст */
--success: #10B981            /* Успех */
--error: #EF4444              /* Ошибка */

/* Glassmorphism */
--glass-bg: rgba(255, 255, 255, 0.05)
--glass-border: rgba(255, 255, 255, 0.1)
--glass-blur: 20px
```

```css
/* Шрифты */

--heading-1: 30px / Bold (700)

/* Межстрочный интервал */

```

### Отступы и радиусы

```css
/* Радиусы скругления */
--radius-sm: 8px      /* Мелкие элементы */
--radius-md: 12px     /* Средние элементы */
--radius-lg: 16px     /* Карточки */

/* Сенсорные цели */

```

## 🎯 Ключевые принципы дизайна

### Glassmorphism

✅ **Полупрозрачный фон** с размытием (backdrop-filter: blur)
✅ **Тонкие границы** с низкой непрозрачностью
✅ **Многослойность** - элементы накладываются друг на друга
✅ **Градиенты** для акцентов и hero-элементов

### Accessibility

✅ **Контрастность** минимум 4.5:1 для текста (WCAG AA)
✅ **Сенсорные цели** минимум 44x44px
✅ **Читаемость** - крупные шрифты (16px+)

```jsx
const navigate = useNavigate()
const location = useLocation()

// При переходе к деталям сохраняем state
navigate(`/documents/${doc.id}`, { 
  state: { previousFilter: category } 
})

// При возврате восстанавливаем фильтр
const handleBack = () => {
  const savedFilter = location.state?.previousFilter || 'all'
  setCategory(savedFilter)
  navigate('/documents')
}
```

### 🎯 Ключевые особенности реализации

✅ **Реактивность:** Изменение `category` автоматически обновляет список (React useState)

✅ **Сохранение состояния:** Фильтр остаётся активным при возврате из деталей

✅ **Клиентская фильтрация:** Не требуется повторный запрос к API при смене категории

✅ **Единый источник данных:** Все документы загружаются один раз, фильтруются локально

✅ **Оптимизация:** Используется `useMemo` для кеширования отфильтрованного списка

```jsx
const filteredDocs = useMemo(() => {
  return documents.filter(doc => 
    category === 'all' || doc.file_type === category
  )
}, [documents, category])
```
